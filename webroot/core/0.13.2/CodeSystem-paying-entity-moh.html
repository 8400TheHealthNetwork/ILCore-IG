CodeSystem: Payer (Paying Entity / גורם משלם)

Official URL: http://fhir.health.gov.il/cs/paying-entity-moh Version: 0.13.2
Draft as of 2023-12-23 Computable Name: PayingEntityMoH

Codes for different paying entities in Israel; including HMOs, IDF, MoD, PLA, Prison system

This Code system is referenced in the content logical definition of the following value sets:

  • IsraelHMO
