Extension: Ext: Hebrew Date

Official URL: http://fhir.health.gov.il/StructureDefinition/ext-hebrew-date Version: 0.13.2
Draft as of 2023-12-23 Computable Name: HebrewDate

Extension: Allows the use of Hebrew dates

Context of Use

This extension may be used on the following element(s):

  • Element ID date
  • Element ID dateTime

Usage info

Usage:

  • Examples for this Extension: Patient/patient-with-hebrew-date and Patient/patient-with-two-addresses

Summary

Complex Extension: Extension: Allows the use of Hebrew dates

  • display: string: Full display of Hebrew date
  • day: CodeableConcept: Code for Hebrew day of month
  • month: CodeableConcept: Code for Hebrew month
  • year: string: Hebrew year
