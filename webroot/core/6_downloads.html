ניתוח R4-R4B

This is an R4 IG. None of the features it uses are changed in R4B, so it can be used as is with R4B systems. Packages for both R4 (health.gov.il.fhir.core.r4) and R4B (health.gov.il.fhir.core.r4b) are available.
