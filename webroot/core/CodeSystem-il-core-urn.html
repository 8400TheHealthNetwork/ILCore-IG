CodeSystem: CodeSyestem for URNs

Official URL: urn:ietf:rfc:3986 Version: 0.13.2
Draft as of 2023-12-23 Computable Name: ILCoreURN

Codes for for URNs of different kinds, instead of using urn:ietf:rfc:3986 and urn:iso:std:iso:3166 as CodeSystem identifiers

This Code system is referenced in the content logical definition of the following value sets:

  • PassportUri
  • StatisticalAreaUri
  • ConsentContentClass
