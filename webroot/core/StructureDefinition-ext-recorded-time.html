Extension: Ext: ILCore Recorded Time

Official URL: http://fhir.health.gov.il/StructureDefinition/ext-recorded-time Version: 0.13.2
Draft as of 2023-12-23 Computable Name: ILCoreRecordedTime

Extension: The time in which the medication order / administration / dispense was recorded

Context of Use

This extension may be used on the following element(s):

  • Element ID MedicationRequest
  • Element ID MedicationAdministration
  • Element ID MedicationDispense

Usage info

Usage:

  • Use this Extension: ILCore MedicationAdministration Profile and ILCore MedicationRequest Profile
  • Examples for this Extension: MedicationAdministration/morphine-hospitalization
