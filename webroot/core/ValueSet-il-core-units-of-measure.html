ValueSet: IL Core Units Of Measure Codes

Official URL: http://fhir.health.gov.il/ValueSet/il-core-units-of-measure Version: 0.13.2
Draft as of 2023-12-23 Computable Name: ILCoreUnitsOfMeasure

A list of codes for Units Of Measure for the IL Core. Note: UCUM code SHALL be used, unless one cannot be found, in which case a SNOMED-CT code SHALL be used

References

This value set is not used here; it may be used elsewhere (e.g. specifications and/or implementations that use this content)

Logical Definition (CLD)

This value set includes codes based on the following rules:

  • Include all codes defined in http://unitsofmeasure.org
  • Include codes from http://snomed.info/sct where concept descends from 408102007 (Unit dose (qualifier value))
  • Include codes from http://snomed.info/sct where concept descends from 732935002 (Unit of presentation (unit of presentation))

 

Expansion

Expansion based on:

  • codesystem UCUM version 2.0.1
  • system UCUM version 2.0.1
